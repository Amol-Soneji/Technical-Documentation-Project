Introduction

C++ is an object-oriented programming language that is used for software development. C++ utilizes functions, classes, and inheritance to create object-oriented code. C++ has a large standard library.

Semantics and Code Structure

C++ follows a standard flow of code, where certain items are to be placed at specific locations. Bellow is an example of a Hello World program in C++.

#include <iostream>
int main()
{
 std::cout << "Hello World!";
}

The first line is used to tell the compiler that the program will need to use the Input/Output stream standard library. The next line is a function declaration. Here the "main" function is being declared. The main function is required in every C++ program as that is where the program starts executing. Following the opeing and closing curly braces "{}" is the body of the function. Here we are stating to send the string "Hello World!" to the cout function in the iostream standard library. The cout function is used to print data to the console.

Variables

Variables are used to store data via a name in the program. There are many types of variables. These types are called data types, and each of them have a certain size limit in number bits of data they can hold. The most common data types are the following.:

  • "char" - It is 1 byte (8 bits) in size. This type is used with characers.
  • "un/signed char" - This is 1 byte (8 bits), and is used with numbers.
  • "int" - 2 bytes (16 bits)
  • "long" - 32 bits
  • "bool" - 1 bit (is used to designate true or false).
  • "void" - Is not used for storage, but is usually used with functions that do not return anything.

For decimal numbers, there are two other data types that are used:

  • "double" - Max precission of 15-16 significant figures.
  • "float" - Max precission of 6-7 siginificant figures.

Note that unlike other programing languages like Java, the strings are not a default data type, and require the use of the string library by putting #include <string> at the begining of the program.

Variables can be declared by first putting the type of the variable followed by the name of the variable. For example, int a; declares the variable "a", to be an interger. To set a value to the variable, we do a = 3; and this will set a to the value 3.

Conditionals and Flow Control

Conditionals are used to see if a conditional statment is true or false, and thus accordingly continue executing in a certain maner. There are 5 major conditional statements used for the flow controll of the program.

if, else if, and else:

If one wants a certain portion of code to only be used if a specific condition is met, the if statment can be used. Note that for any conditional statements, where the command(s) to be executed if a specific conditon is met, is longer than one line, then it must be placed inside curly braces "{}". For example, if we wanted to print 3 if a is equal to 3 we would do the following:
if(a == 3)
 std::cout << "3";
 What if one wanted to print no only if the above statement is not true. In that situation we would use the else statement. For example:
if(a == 3)
 std::cout << "3";
else
 std::cout << "No";
 If one wanted to also print "negative" only if the variable is set to a negative number, in the above example we would then use the "else if" as shown bellow:
if(a ==3)
 std::cout << "3";
else if(a < 0)
 std::cout << "Negative";
else
 std::cout << "No";

while and do while loops:

There are two types of while loops in C++. While loops in general are used for tasks that must be done repetitively. For example, lets says there is an int b = 5; and someone wanted to subtract 1 from it until it reaches 0, and print the value of b at each repitition. This can be done by the folloing method:
while(b != 0)
{
 std::cout << b;
 b--;
}
 This can be done also via a do while loop.
do
{
 std::cout << b;
 b--;
} while(b != 0);

for loops:

Unlike while loops, for loops can combine repetitive increments, as well as declaring an incremental variable with a scope on in the loop, and a conditional of when to stop the loop. For loops can be used for anything that requires looping, but is seen a lot when c++ vectors or arrays are used. For example, if one wanted to create a interative value of b, and wanted to print the value of b each time it is subtracted until it equalls a, one can do the following:
for(int b = 5; b != a; b--)
{
 std::cout << b;
}

switch statements:

C++ switch statements are similar to if statements, but utilise a variable to determine which case statement to execute. This is helpful if there are lots of possiblities, where using multiple else-if statements would make the code too cluttered. Unlike others, the code inside each case does not need curly braces {} if its more than one line. Also, a case of default can be used if there is an invalid value for the variable used in the switch. Default case usually always execute unless a break; is stated in the case, which causes the program to jump of of the witch statement block during execution. For example, if one wanted to display the name of a month based on a number from 1-12, it can be done the following way:
switch(month)
{
 case 1:
  std::cout << "January";
  break;
 case 2:
  std::cout << "February";
  break;
...
 case 12:
  std::cout << "December";
  break;
 default:
  std::cout << "Invalid number for month. ";
}

Functions

Functions in C++ have a specific layout, and placement of the functions must also be taken into account in order to compile and execute. For example, bellow is the implementation of the "main" function that counts to 10.
int main()
{
 for(int i=0; i<11; i++)
 {
  std::cout << i;
 }
}
 The first of the function is the declaration of the function. The first part of it is the return type of the function. Here it is "int". The return type is the data type of the value that is returned to the calling function. If nothing is to be returned, the return type is then to be set to "void". After the return type comes the name of the function. The name of the function is used by another function that wants to call the function. After the function name comes a parenthesis "()". Items in the parenthesis are called parameters. Parameters are variables or pointers that are passed to the function. It is required to indicate the type of the variable or pointer type that is being passed to the parameter. For example, if a function was to have a parameter consisting of and interger and a pointer to a bool, the function would be declared as follows:
int main(int a, bool* b){...}

In C++, functions must be implmented before they are called, otherwise there will be a compilation error. Having multiple functions for different tasks the program must do, makes it easier to read and understand the code. It also helps when trouble shooting the code, and is also an important aspect when it comes to object-oriented programming. For example, if one wanted to create a basic calculator program, the following would be the structure of the function layout of the program:
int add(int a, int b){...}
int subtract(int a, int b){...}
int multiply(int a, int b){...}
double divide(int a, int b){...}
int main(){...}

Arrays

Arrays are like lists that can contain multiple values of the same data type. However, the size of the array is fixed, and can't be increased. There may be fewer elements than the size of the array, but the rest of the elements will be set to 0 by default. The following is an example of and int array of size 7 being created with 3 predefined values:
int a[7] = {6, 18, 1};
 Note that the other 4 elements are implicitly set to 0. Also note that the index of the array starts at 0 rather than 1. Thus the last index of the array a would be 6.

To add or change the value of an element in an array, one can do the following:
a[3] = 10;
 To set another vairable for example an integer b, to a value in the array, this can be done by:
b = a[3];
 This sets the value of b to 10. Knowing this, it is possible to iterate over an array, and display its values by doing the following:
for(int index = 0; index < 7; index++)
{
 std::cout << a[index];
}

Classes

The most important part of any object oriented programming language is the use of classes. A class is like a blueprint to create an object of of a specific type. For example, one can create a class called "Organism". This organism class can have a name and age. For example:
class Organism
{
 std::string name;
 int age;
 public:
  Organism(std::string name, int age);
  void displayInfo();
}
Organism::Organism(std::string name, int age)
{
 this->name = name;
 this->age = age;
}
void Organism::displayInfo()
{
 std::cout << "Name: " << name << " Age: " << age;
}
 The first part is the declaration of the class. What follows the "class" keyword is the name of the class. The next part is the stating the private class variables. These variables are only accessable by any function that is part of the class. The next part is the public class variables as well as any class functions that are designated to be accessable to functions outsie of the class. The first public function in the example above is called the constructor. This type of function is used to initialize and create the class.

The public functions of the class are implmented in specific manner. This is done by name of class::function name.

To create an object of the class and print its info, one can do the following:
Organism organ("Human", 20);
organ.displayInfo();
 Here "organ" is a variable of the type Organism. In the first line the Organism constructor is being utilized to create a instance of the Organim class, and set the values of the private variables. The second line is how to access the displayInfo function of the class. To access functions in a class after setting the variable to the type of the class, it is done by variable.functionName().
